# Changelog

本文档记录 MacExplorer 的重要产品和工程变更。

## Unreleased

### Added

- 新增 MIT License、贡献说明、安全说明、GitHub issue/PR 模板和基础 Swift 构建 workflow，为 GitHub 开源做准备。
- 新增 release manifest 生成脚本，并接入本地发布验证流程。
- 新增 `VERSION` 文件，让 app 版本号和 build 号从打包脚本中解耦。
- 新增发布流程文档，记录版本更新、验证、打 tag 和 GitHub Release 上传步骤。
- 发布验证脚本新增依赖命令预检，减少换机器时的模糊失败。
- 新增仓库卫生检查脚本，并接入发布验证流程。
- 新增多窗口入口、窗口尺寸位置恢复和标签页右键管理。
- 打包脚本支持生成 `arm64`、`x86_64` 和 `universal` 架构包。
- 新增永久删除入口，支持二次确认、操作中心进度和 `Option+Command+Delete` 快捷键。
- 详情面板新增异步文件夹统计，显示递归文件数、子文件夹数、总大小和跳过的不可访问目录数。
- 新增打开方式入口，可从菜单栏或文件右键菜单选择其他应用打开所选项目。
- 侧边栏 Locations 区块改为监听系统卷挂载、卸载和重命名事件，外接盘、DMG 等卷变化无需重启应用即可刷新。
- 收藏夹支持从文件区或系统拖入目录添加，空收藏夹状态也保留可拖放目标。
- 侧边栏宽度支持 220 到 280 pt 拖拽调整，并会保存到下次启动。
- Display 菜单和工具栏新增排序控制，可切换排序字段、升降序和文件夹优先，并立即刷新列表或递归搜索结果。
- 新增目录级视图和排序偏好，不同文件夹会记住自己的列表/图标视图、排序字段、升降序和文件夹优先设置。

### Fixed

- 修复文件拖到文件夹项时仍停留在原目录的问题，列表视图和图标视图的目录项现在都能作为拖放目标。
- 补齐 Windows 习惯快捷键：`F2` 重命名，文件区 `Delete` 移到废纸篓。
- 最近访问目录改为仅保留最近 5 个有效文件夹，侧边栏更简洁。
- 修复文件列表和图标视图中单击、双击手势可能互相抢占，导致双击文件夹不进入目录的问题。
- 启动冒烟脚本支持指定要验证的 `.app` 路径，便于验证 x86_64 和 universal 包。
- 修复递归搜索在切换目录、切换筛选或取消旧任务时可能出现空结果、错误加载状态或旧任务覆盖新状态的问题。
- 当前目录搜索和递归搜索统一使用分词、大小写和音调不敏感的匹配规则。
- 递归搜索会匹配结果所在的相对路径，搜索父文件夹名或路径片段也能找到对应项目。
- 筛选条件激活但搜索框为空时，空状态和状态栏会正确显示为搜索/筛选结果状态。
- 递归搜索遇到无权限或不可读取目录时会继续搜索，并在状态栏提示跳过的位置数量。
- 扩展名筛选支持用空格、逗号或分号输入多个扩展名。
- 递归搜索结果在列表视图和图标视图中显示所在位置，方便定位原始路径。
- 递归搜索改为渐进刷新结果，并在状态栏显示已扫描数量。
- 递归搜索增加结果数量上限提示，避免大目录或宽泛筛选导致列表和内存失控。
- 搜索结果出现后不再被中央加载指示器遮挡。
- 进入新目录、切换标签或使用后退/前进时会清理当前搜索上下文，避免旧搜索词继续过滤新位置内容。
- 文件操作会校验空名称和包含斜杠的名称，避免新建或重命名落到异常路径。
- 复制/移动文件夹时会阻止将文件夹放入它自身或子文件夹中。
- 替换同名项目时先用临时备份完成替换，降低复制/移动失败导致原目标丢失的风险。
- 带点号的文件夹重复命名会在完整文件夹名后追加 `copy`，不再误当成扩展名处理。
- 文件操作运行期间会禁用新建、重命名、删除、粘贴和拖放接收，避免操作中心状态被并发任务覆盖。
- 操作中心只对可取消的复制/移动任务显示取消入口，取消后等待后台任务退出再允许新的文件操作。
- 目录加载增加请求标识和标签页校验，快速切换标签、刷新或导航时旧加载结果不会覆盖当前文件列表。
- 文件图标显示接入内存缓存，减少大目录滚动、选择和刷新时重复请求系统图标。
- 详情面板显示文件可读、可写、可执行权限摘要，并支持中英文文案。

## 0.1.0 - 2026-07-06

首个可本机运行和打包验证的版本。

发布说明：[docs/releases/v0.1.0.md](./docs/releases/v0.1.0.md)

### Added

- 建立 SwiftPM 原生 macOS 应用工程。
- 实现 Windows Explorer 风格主窗口、标签栏、工具栏、地址栏、侧边栏、文件区、详情/预览面板和状态栏。
- 实现真实目录读取、详情列表视图、图标网格视图、系统文件图标、排序、显示隐藏文件和显示/隐藏扩展名。
- 实现返回、前进、上一级、刷新、面包屑路径、可编辑地址栏和路径自动补全。
- 实现标签页恢复、标签页拖拽排序和独立导航历史。
- 实现快速访问、收藏夹、最近访问、系统位置侧边栏。
- 收藏夹支持持久化、右键管理、拖拽排序、上移和下移。
- 实现当前目录搜索、递归搜索、搜索取消和扩展筛选条件。
- 实现新建文件夹、新建空文件、复制、剪切、粘贴、重命名、移到废纸篓。
- 实现文件操作冲突处理、进度状态、操作中心、历史记录和复制/移动取消。
- 接入 Quick Look，支持 Space 预览、工具栏预览和右侧内嵌预览。
- 实现文件拖拽导出和中央文件区拖放接收。
- 实现键盘多选、Shift 范围选择、Cmd+A、方向键选择、Enter 打开和焦点快捷键。
- 默认中文界面，支持 English 切换。
- 新增应用图标生成脚本，并生成 `.iconset` / `.icns` 资源。
- 新增 release `.app` 打包脚本、release zip / dmg 归档脚本和本地发布验证脚本。
- 新增 README、开发计划和发布检查清单。

### Changed

- 开源仓库中的发布产物精简为仅保留当前最新 DMG 安装包和 SHA256 校验文件。
- 工具栏隐藏文件按钮使用 `eye.slash`，避免和 Quick Look 的 `eye` 图标混淆。
- 扩展名显示开关使用文档图标，避免中文环境下 `textformat.abc` 显示成“甲乙丙”。
- 文件区保留键盘焦点能力，但不显示大面积蓝色焦点框。
- 应用图标升级为更立体的 macOS 圆角底座、窗口和蓝色文件夹组合。

### Known Limitations

- 当前包使用本地 ad-hoc 签名，尚未配置 Developer ID 签名和 notarization。
- 当前 Command Line Tools 环境未完整暴露 `XCTest` / Swift `Testing` 所需模块，暂未保留会失败的 test target。
- Gatekeeper 对本地 ad-hoc 签名 app 的评估会提示 rejected，这是对外分发前的预期限制。
