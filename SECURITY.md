# Security Policy

MacExplorer 是本地文件管理器，文件读写、删除、拖拽和搜索逻辑都属于高敏感区域。

## 报告安全问题

如果你发现可能导致数据丢失、越权访问、意外删除、路径穿越、凭据泄漏或任意命令执行的问题，请先不要公开完整利用细节。

推荐做法：

- 在 GitHub 创建一个标题以 `[Security]` 开头的 issue，只描述影响范围和联系方式。
- 不要在公开 issue 中附带可直接利用的 PoC、真实隐私路径、token、证书或用户数据。
- 维护者确认后，再通过双方认可的私下渠道补充细节。

## 支持范围

当前优先处理：

- 文件操作可能造成非预期覆盖、删除或移动。
- 搜索、拖放、地址栏跳转中的路径边界问题。
- 发布包、脚本或仓库中出现凭据、证书或敏感本地路径。
- 可能绕过 macOS 权限预期的行为。

当前已知限制：

- 发布包仍使用本地 ad-hoc 签名，尚未接入 Developer ID 签名和 notarization。
