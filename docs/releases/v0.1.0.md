# MacExplorer v0.1.0

发布日期：2026-07-06

这是 MacExplorer 的第一个可本机运行、可打包验证的版本。它已经具备 Windows Explorer 风格文件管理器的主要浏览、导航、搜索、文件操作、预览、标签页、收藏夹和中英文切换能力。

## 发布产物

开源仓库仅保留当前最新 DMG 安装包和对应 SHA256 校验文件：

```text
release-artifacts/MacExplorer-0.1.0-22-macos.dmg
release-artifacts/MacExplorer-0.1.0-22-macos.dmg.sha256
release-artifacts/MacExplorer-0.1.0-22-macos-x86_64.dmg
release-artifacts/MacExplorer-0.1.0-22-macos-x86_64.dmg.sha256
release-artifacts/MacExplorer-0.1.0-22-macos-universal.dmg
release-artifacts/MacExplorer-0.1.0-22-macos-universal.dmg.sha256
```

本地完整打包输出位置：

```text
dist/MacExplorer.app
dist/MacExplorer-0.1.0-22-macos.zip
dist/MacExplorer-0.1.0-22-macos.zip.sha256
dist/MacExplorer-0.1.0-22-macos.dmg
dist/MacExplorer-0.1.0-22-macos.dmg.sha256
dist/MacExplorer-0.1.0-22-macos.manifest.json
dist/MacExplorer-0.1.0-22-macos-x86_64.dmg
dist/MacExplorer-0.1.0-22-macos-x86_64.dmg.sha256
dist/MacExplorer-0.1.0-22-macos-universal.dmg
dist/MacExplorer-0.1.0-22-macos-universal.dmg.sha256
```

SHA256 以同目录下的 `.sha256` 文件为准。每次重新打包后，使用以下命令校验：

```bash
cd dist
shasum -a 256 -c MacExplorer-0.1.0-22-macos.zip.sha256
shasum -a 256 -c MacExplorer-0.1.0-22-macos.dmg.sha256
```

## 验证结果

已通过：

- `swift build`
- `swift build -c release`
- `scripts/package_app.sh release`
- `codesign --verify --deep --strict --verbose=2 dist/MacExplorer.app`
- zip SHA256 校验
- `hdiutil verify dist/MacExplorer-0.1.0-22-macos.dmg`
- release manifest JSON 校验
- dmg SHA256 校验
- `scripts/smoke_launch_app.sh`
- `scripts/verify_release.sh`

预期限制：

- `spctl --assess --type execute --verbose=4 dist/MacExplorer.app` 当前会提示 rejected，因为这个版本仍使用本地 ad-hoc 签名，尚未进行 Developer ID 签名和 notarization。

## 主要功能

- 原生 SwiftUI + AppKit macOS 应用。
- 文件详情列表、图标网格视图、统一排序菜单和目录级视图/排序偏好。
- 地址栏、面包屑路径、返回、前进、上一级、刷新。
- 标签页、标签页恢复和拖拽排序。
- 快速访问、收藏夹、最近访问、系统位置、动态外接卷和可拖拽宽度侧边栏。
- 收藏夹持久化、右键管理、拖入目录添加和拖拽排序。
- 当前目录搜索、递归搜索和搜索筛选。
- 递归搜索支持渐进刷新、扫描数量反馈和结果上限提示。
- 切换目录或标签时会清理搜索上下文，避免旧搜索条件影响新位置。
- 新建、复制、剪切、粘贴、重命名、移到废纸篓、永久删除和打开方式选择。
- 文件操作冲突处理、操作中心和复制/移动取消。
- Quick Look、右侧详情/预览面板和异步文件夹统计。
- 文件拖拽导出和文件区拖放接收。
- 默认中文界面，支持 English 切换。
- 自定义应用图标、release zip 和 release dmg。
